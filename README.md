# Ardu-Lights test project

Simple project to display content to a RGB LED Matrix. This is the front-end repo

## Stack & Hardware

- Arduino ESP8266 board 
- 8x8 RGB W2812b LED Matrix
- MongoDB
- NodeJS
- Aedes MQTT Server
- React
